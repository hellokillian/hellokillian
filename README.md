### hi, i'm killian

i build [open interpreter](https://github.com/OpenInterpreter/open-interpreter) — a natural language interface for your computer. tell it what you want in plain english and it writes + runs code to make it happen. runs locally, no cloud, your data stays yours.

currently working on **[interpreter](https://openinterpreter.com)**, a desktop app that fills out PDFs, edits spreadsheets and word docs, and picks up new skills as you use it. all offline, all free.

i've also been building **[praison](https://github.com/PraisonLabs/Praison)** for the past year — a production-ready multi-agent framework where AI agents collaborate, self-reflect, and teach themselves new capabilities. supports 100+ LLMs, works with code or no-code, and handles everything from RAG to async workflows.

i think the best AI is the kind that lives on your machine and just quietly does things for you.
